üìã Project Overview

This project delivers a comprehensive analysis of social media advertisement performance across multiple platforms including Facebook, Instagram, and TikTok. The dashboard provides actionable insights into campaign effectiveness, audience engagement patterns, and budget allocation strategies.

The analysis encompasses three key dimensions: audience demographics and behavior, overall campaign performance metrics, and temporal trends in engagement and budget utilization. This multi-faceted approach enables data-driven decision-making for optimizing social media advertising strategies.

Problem Statement

Social media advertising generates vast amounts of data across multiple platforms, making it challenging to extract meaningful insights. Marketing teams need to understand which demographics respond best, when to post for maximum engagement, and how to allocate budgets effectively across campaigns and platforms.

Solution

Created an interactive Power BI dashboard with three specialized pages that consolidate social media advertising data, providing real-time visibility into audience insights, campaign performance, and engagement trends with drill-down capabilities for detailed analysis across demographics, geography, and time periods.

üîß Technical Implementation

Data Pipeline

  • Integrated data from multiple social media platforms including Facebook, Instagram, and TikTok APIs
  • Implemented ETL processes using Power Query for cleaning and standardizing cross-platform advertising data
  • Created a star schema data model linking campaign, audience, location, and time dimensions
  • Set up automated data refresh schedules to capture real-time advertising performance metrics

Analysis Techniques

  • Time series analysis to identify peak engagement days (Wednesday) and hours (20:00-21:00)
  • Demographic segmentation analysis revealing optimal audience targeting by age groups and gender
  • Geographic visualization using map visuals to identify high-performing regions and user distribution
  • Budget optimization analysis tracking adjusted budget (578.08K) vs average budget (48.17K) across campaigns
  • Platform performance comparison using stacked bar charts and tables for granular ad type analysis
  • DAX calculations for complex metrics including engagement rates, cumulative trends, and time-slot performance
